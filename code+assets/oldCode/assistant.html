Coffee brewer

I created this coffee brewing machine as a three month senior studio school project. The prompt was to design a product for injection molding that uses a DC motor to automate a common task. I set a goal for myself to learn the basics of electrical engineering by integrating the Arduino ecosystem with my prior experience in mechanical design. This is a current project of mine and I hope to have the second prototype in the near future.
